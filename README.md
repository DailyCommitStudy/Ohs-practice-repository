### 2. TIL (Today I Learned) 새롭게 공부한 내용 정리! < Continue >

그 외에도 데이터분석 관련 유튜브를 보면서 새롭게 안 내용 추가 정리.   
해당 파일 저장 시 커밋 메세지에 add (새롭게 안 내용) 라고 추가해서 기재할 예정.
